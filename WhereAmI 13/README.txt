Game  extended instructions:
▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬

Execute the main in the RUN file to play the game.
——————————————————————————————————————————————————————————————————————————————————————————————————————————————

○ The difficulties dictate the type of images that will be shown, with harder to guess images being considered 
  harder and on hard you have 30 seconds to make a guess.

○ An image will be displayed and you need to click the region of the map where you think the image is from.

○ Based on the distance from the guess to the actual location a score will be given.

○ The game consists of 10 rounds.

○ If screen is blank wait a short while and the images should load.

——————————————————————————————————————————————————————————————————————————————————————————————————————————————
To start a new game close the window and run the program again.
